Nmap

Nmap ("Network Mapper") is a free and open source utility for network discovery and security auditing.

W3AF

w3af is a complete environment for auditing and exploiting Web applications.

Tenable Nessus

Tenable Nessus is a vulnerability scanner that helps identify and fix security issues in devices, applications, operating systems, and other network resources.

OnSecurity

OnSecurity is a leading CREST-accredited penetration testing vendor based in the UK, dedicated to delivering high-impact, high-intelligence penetration testing.

Wireshark

Wireshark is a free, open-source network protocol analyzer that captures and displays network traffic in real time.

Owasp Zap

OWASP ZAP (Zed Attack Proxy) is a free, open-source tool that helps identify and fix security vulnerabilities in web applications

Github Nikto

Nikto is a free software command-line vulnerability scanner that scans web servers for dangerous files or CGIs, outdated server software and other problems.

Cobalt

The Cobalt Platform brings together the data, technology, and talent to resolve security challenges in modern web applications, mobile applications, networks, AI/LLM apps, and APIs.

Rapid7

The Rapid7 Command Platform collects data from across your environment, making it easy for teams to manage vulnerabilities, monitor for malicious behavior, investigate and shut down attacks, and automate your operations.

Astra Security

Astra Security is a cyber security SaaS company that makes otherwise chaotic penetration tests a breeze with its one of a kind Pentest Platform. Astra's continuous vulnerability scanner emulates hacker behavior to scan applications for 9300+ security tests.
